📊 How Costs Are Calculated — Examples

Example 1: Casual Investor — ₹1L trade, 1 stock, once a month

You buy ₹1,00,000 worth of Reliance shares and sell them a month later. One buy, one sell, one stock (ISIN).

Dhan: Brokerage ₹0 + DP (₹12.50 × 1.18 GST × 1 sell × 1 ISIN) = ₹14.75/trade → ₹177/yr (12 trades)
Zerodha: Brokerage ₹0 + DP (₹13.00 × 1.18 × 1 ISIN) = ₹15.34/trade + ₹300 AMC → ₹484/yr
Groww: Brokerage (min(₹20, ₹50) = ₹20 × 2 sides = ₹40) + DP (₹20 × 1.18 × 1 sell × 1 ISIN = ₹23.60) = ₹63.60/trade → ₹763/yr

✓ Dhan saves ₹307 vs Zerodha and ₹586 vs Groww annually.

Example 2: Active Trader — ₹2L trade, 3 stocks, weekly

You trade ₹2,00,000 across 3 different stocks every week (52 round trips/year). You sell each stock separately (3 sell orders).

Dhan: DP (₹14.75 × 3 sells × 3 ISINs) = ₹132.75/trade → ₹6,903/yr
Zerodha: DP (₹15.34 × 3 ISINs) = ₹46.02/trade + ₹300 AMC → ₹2,693/yr
Groww: Brokerage (₹20 × 2 = ₹40) + DP (₹23.60 × 3 sells × 3 ISINs = ₹212.40) = ₹252.40/trade → ₹13,125/yr

⚠ With multiple ISINs and sell orders, DP charges dominate. Dhan's DP multiplies by both sell orders AND ISINs, making Zerodha cheaper for multi-stock active traders.
